# Support

Use GitHub Issues for reproducible bugs and feature requests. Before filing:

1. read the README and runtime recovery guide;
2. confirm the issue still occurs on the latest `main` branch;
3. run `pnpm check:opensource`, `pnpm lint`, `pnpm check-types`, and `pnpm build` for development issues;
4. remove all credentials, local paths, account details, QR codes, and unpublished media.

Platform creator pages are third-party services and can change without notice. Maintainers cannot
restore platform accounts, resolve moderation decisions, or bypass authentication and risk controls.
For security-sensitive reports, follow [SECURITY.md](SECURITY.md) instead of opening a public issue.
